LẤY UDID NGAY
⚠️ HƯỚNG DẪN QUAN TRỌNG:
1. Tải profile bằng nút trên
2. Vào Cài đặt → Cài đặt chung → VPN & Quản lý thiết bị
3. Chọn profile "Lấy UDID Thiết Bị" → Nhấn "Cài đặt"
4. Nhập mật khẩu thiết bị → Nhấn "Cài đặt" lần nữa → "Xong"
5. QUAY LẠI trang này để xem UDID
